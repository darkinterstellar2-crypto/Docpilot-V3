  • • New accounts start with zero access — they can't see anything until you grant it here.
  • • Master toggle ON = sees all projects by default. Use project toggles to block specific ones.
  • • Master toggle OFF = sees NO projects. Use project toggles to grant specific ones.
  • • Module toggles control which tabs are visible within a project.
  • • Only you (superadmin) can create, delete, or manage projects.
