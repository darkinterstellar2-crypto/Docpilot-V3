# Internationalization (i18n)

**File:** `src/js/i18n.js` (242 lines)

DocPilot supports two languages: **English** and **German**. Language toggling is handled client-side with no server involvement.

## How It Works

1. Language preference stored in `localStorage('lang')`, defaulting to `'en'`
2. HTML elements use `data-i18n="key"` attributes
3. On page load, `I18N.init()` replaces text content based on the current language
4. A toggle button is injected into the page header

## Usage in HTML

```html
<!-- Text content -->
<span data-i18n="nav.hub">Hub</span>

<!-- Placeholder -->
<input data-i18n-placeholder="table.search" placeholder="Search...">

<!-- Title attribute -->
<button data-i18n-title="files.upload" title="Upload">📤</button>
```

## Translation Keys

The `I18N.translations` object contains all translation strings organized by section:

| Section | Example Keys |
|---------|-------------|
| `nav.*` | `nav.hub`, `nav.admin`, `nav.profile`, `nav.back` |
| `hub.*` | `hub.projects`, `hub.newProject`, `hub.openDashboard` |
| `dash.*` | `dash.title`, `dash.modules`, `dash.aufmass`, `dash.files`, `dash.einblasenDesc` |
| `auth.*` | `auth.login`, `auth.register`, `auth.email`, `auth.password` |
| `profile.*` | `profile.title`, `profile.changePassword`, `profile.save` |
| `files.*` | `files.title`, `files.upload`, `files.newFolder`, `files.trash` |
| `table.*` | `table.save`, `table.addRow`, `table.search`, `table.export` |
| `mod.*` | `mod.done`, `mod.pending`, `mod.upload`, `mod.markAppointment` |
| `admin.*` | `admin.title`, `admin.users`, `admin.approve`, `admin.permissions` |
| `newProject.*` | `newProject.title`, `newProject.name`, `newProject.create` |
| `common.*` | `common.loading`, `common.error`, `common.confirm`, `common.cancel` |

## API

```javascript
// Get current language
I18N.lang();  // 'en' or 'de'

// Set language (persists + re-applies)
I18N.setLang('de');

// Translate a key
I18N.t('dash.title');  // 'Dashboard' (en) or 'Dashboard' (de)

// Apply all translations to DOM
I18N.apply();
```

## Toggle Button

`I18N.injectToggle()` adds a language toggle button (showing "EN" or "DE") to the page header. Clicking it switches between English and German and re-applies all translations.

## Limitations

- Only English and German are supported
- Not all text is translatable — some strings are hardcoded in JavaScript
- Server-side responses (error messages, log entries) are always in English
- Module-specific labels in the Aufmass schema are not translated (they come from the data file)
